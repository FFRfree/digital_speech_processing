数字语音处理实验

实验一  Matlab语音处理基本指令
    读load，read，write，plot，strip plot，spectrum，sample conversion，etc

实验二  语音信号基本时频分析
    一帧语音信号的窄带，宽带频谱图，语谱图。Log magnitude域
    清音，浊音的时域、频谱域分析（画图出来，对比，给出结论）

实验三  Voiced/Unvoiced/silence 检测系统
    基于短时能量、短时过零率这2个基于规则的方法来实现

实验四  MFCC特征提取及分析
    提取一帧浊音信号的MFCC特征（13维的倒谱参数），并对其做PCA聚类分析

实验五  基于K-means的语音、噪声分类（男女声分类）
    采用MFCC为特征，用K-means做聚类来做男女声分类
